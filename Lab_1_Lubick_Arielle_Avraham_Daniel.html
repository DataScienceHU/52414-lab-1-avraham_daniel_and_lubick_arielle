  1. Add a new column named Diff to both the cases.agg and the deaths.agg data-frames. This new column should register the daily Value difference for each country. In other words, the Diff column shows how many new cases/deaths each country incurs every day. Hint - diff must be per country. (7pts)
  2. Find the top 10 instances of country and date combinations with the greatest absolute number of new daily Corona cases and deaths (separately). Print the result in a descriptive format. (5pts)
  3. In one figure, plot Italys new daily Corona cases AND deaths as a function of Date. Choose the plot type you think that makes the most sense. (3pts)
  4. Plot the same graph as in (c), but this time plot the number of new cases on the logarithm scale. What can we learn? (5pts)

### part a
### making the Diff for cases.agg
cases.agg$Diff <- rep(0)
cases.agg$Diff[1] <- cases.agg$Value[1]
for (i in 2:length(cases.agg[,1])){
  if (cases.agg$Country.Region[i]==cases.agg$Country.Region[i-1]){
    cases.agg$Diff[i]= cases.agg$Value[i]-cases.agg$Value[i-1]
